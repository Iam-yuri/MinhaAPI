## Quem tiver interesse no projeto, e quiser da uma olhada:

### 1. Clone este repositório:
   ```bash
   git clone https://github.com/Iam-yuri/MinhaAPI
   ```
### 2. Navegue até o diretório do prjeto:
   ```
   cd MinhaAPI
   ```
### 3. Restaure as dependências:
   ```
   dotnet restore
   ```
### 4. Execute o projeto:
   ```
   dotnet run
   ```
